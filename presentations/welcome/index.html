Titanium Mobile is a platform for building best in class cross-platform mobile applications in JavaScript

Titanium Mobile

  • Only cross-platform mobile toolkit with native UI and performance
  • Free and open source under the Apache 2.0 license
  • Supports iOS, Android, Blackberry, and Mobile Web

How Titanium Works

  • You write Javascript
  • Titanium maps your JS to native bindings
  • Your JS + JS engine + native bindings = native mobile app

Development Workflow

  • UI and app logic coded in Javascript
  • Write code, manage projects, and distribute with Titanium Studio
  • Target multiple platforms from a single codebase

    var win = Ti.UI.createWindow({
        backgroundColor: '#fff'
    });
    var view = Ti.UI.createView({
        backgroundColor: '#a00',
        height: 200,
        width: 200
    });
    view.addEventListener('click', function(e) {
        alert('clicked');
    });

    win.add(view);
    win.open();
